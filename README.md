## Setup

1. Install dependency:  
```pip install python-dotenv```
2. Create a ```.env``` file with your Gmail App Password:
    ```password=YOUR_APP_PASSWORD```
3. Set your Gmail in the script:
  ```SENDER_EMAIL = "yourgmail@gmail.com"```

4. Add participants in the participants list.

5. Run:
   ```python secret_santa.py```
